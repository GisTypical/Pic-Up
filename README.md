# PIC-UP

> *A place where you can save images anywhere, anytime*

## Technologies

- Frontend
    - React
    - Tailwind
    - React Query
- Backend
    - Python
    - Flask

## Project structure

This project was planned as a *monorepo*:
```
frontend/
├─ package.json
├─ ...
backend/
├─ models/
├─ controllers/
app.py (starts API)
Config.py
requirements.txt 
package.json
```

## Build source

First install corresponding packages

> Backend

```
pip install -r requirements.txt 
```

> Frontend

```
npm run install-front
```
